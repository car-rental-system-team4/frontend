# Frontend Documentation

This folder contains comprehensive documentation for all frontend modules of the Car Rental System.

## Module Documentation

### Core Modules
1. **[FRONTEND_AUTHENTICATION_MODULE.md](./FRONTEND_AUTHENTICATION_MODULE.md)** - Authentication Module
   - React Context API for auth state
   - localStorage for token persistence
   - Axios interceptors
   - Role-based navigation

2. **[FRONTEND_CARS_MODULE.md](./FRONTEND_CARS_MODULE.md)** - Cars Module
   - Vehicle browsing and filtering
   - Car details page
   - Data transformation
   - Image handling

3. **[FRONTEND_BOOKINGS_MODULE.md](./FRONTEND_BOOKINGS_MODULE.md)** - Bookings Module
   - Booking list display
   - Tab-based navigation
   - Booking details modal
   - Cancel booking functionality

4. **[FRONTEND_PAYMENT_MODULE.md](./FRONTEND_PAYMENT_MODULE.md)** - Payment Module
   - Date selection
   - Price calculation
   - Sequential API calls
   - Payment processing

5. **[FRONTEND_PROFILE_MODULE.md](./FRONTEND_PROFILE_MODULE.md)** - Profile Module
   - Profile viewing and editing
   - Password change
   - Account deletion
   - Context synchronization

6. **[FRONTEND_REVIEWS_MODULE.md](./FRONTEND_REVIEWS_MODULE.md)** - Reviews Module
   - Review submission
   - Star rating component
   - Review display
   - Average rating calculation

7. **[FRONTEND_COMPLAINTS_MODULE.md](./FRONTEND_COMPLAINTS_MODULE.md)** - Complaints Module
   - Complaint submission
   - Complaint history
   - Status display

8. **[FRONTEND_ADMIN_MODULE.md](./FRONTEND_ADMIN_MODULE.md)** - Admin Module
   - Admin dashboard
   - Data visualization (Recharts)
   - Search and filtering
   - User/Booking/Vehicle management

9. **[VENDOR_IMPLEMENTATION.md](./VENDOR_IMPLEMENTATION.md)** - Vendor Module
   - Vendor dashboard
   - Vehicle management
   - Booking management
   - Revenue tracking
   - File upload

## Documentation Structure

Each module documentation includes:
- ✅ **Architecture**: Component structure and data flow
- ✅ **Core Concepts**: Detailed explanations of React concepts used
- ✅ **React Concepts**: Hooks, patterns, best practices
- ✅ **Component Implementation**: Code structure and patterns
- ✅ **How It Works**: Step-by-step execution flows
- ✅ **Interview Questions**: Common questions with detailed answers
- ✅ **Code Walkthrough**: Complete flow examples
- ✅ **Best Practices**: React and frontend best practices

## Technology Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Routing**: React Router DOM v6
- **HTTP Client**: Axios
- **State Management**: React Context API
- **Charts**: Recharts
- **Styling**: Bootstrap 5
